# app delete
```bash
$ copilot app delete [flags]
```

## コマンドの概要

`copilot app delete` Application に関連付けられた全てのリソースを削除します。

## フラグ

```bash
-h, --help                          help for delete
    --yes                           Skips confirmation prompt.
```

## 実行例
Application を強制的に削除します。
```bash
$ copilot app delete --yes 
```
